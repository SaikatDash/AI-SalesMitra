## ⚡ Quick Start (5 Minutes)

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/SaikatDash/Medicine-Sales-Analysis-Dashboard-with-Future-Business-Predictions.git
cd AI-SalesMitra
```

### 2️⃣ Create Virtual Environment (Recommended)
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

### 3️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

**Core Requirements:**
```bash
pip install streamlit pandas plotly scikit-learn scipy numpy
```

### 4️⃣ Run the Dashboard
```bash
streamlit run app.py
```

✨ **Dashboard opens at** `http://localhost:8501`

## 🎓 Installation & Setup Guide

### Prerequisites
- Python 3.9 or higher
- pip/conda package manager
- 4GB RAM minimum (8GB recommended)
- 500MB disk space

### Detailed Setup
```bash
# 1. Clone repository
git clone <repo-url>
cd AI-SalesMitra

# 2. Create virtual environment
python -m venv venv
source venv/bin/activate for both frontend and backend folders

# 3. Upgrade pip
pip install --upgrade pip

# 4. Install dependencies
pip install -r requirements.txt for both folder

#5.backend running
go to backend folder and run : uvicorn app:main --reload

# 7. Run application
go to the frontend folder and run the command : python -m streamlit run sample.py

#8. If login is not working then do register home page.
```

### Environment Variables
Create `.env` file:
```env
STREAMLIT_SERVER_HEADLESS=true
MODEL_PATH=./models/
DATA_PATH=./data/
LOG_LEVEL=INFO
```
